HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are contained between a closing and opening tag.
  • Heading tags are numbered from 1 to 6, 1 being the biggest heading & should be used only once in a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 ways to style a webpage using CSS: inline CSS, internal CSS style sheet & external CSS style sheet.
  • Two selectors can be combined into a single CSS rule with a comma, allowing the declarations to affect both sectors.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add [file] adds a file as it currently looks now to your next commit.
  • git log shows all commits in the current branch's history.
